# PromoAjax
Aplicativo de cadastros de promoções de lojas em Spring com Ajax, usando OpenGraph e TwitterCard.<br/>

## Linguagens
* Java<br/>
* JS (jQuery)<br/>
* Html<br/>
* CSS<br/>

## Tecnologias usadas
* Spring<br/>
* Thymeleaf<br/>
* Maven<br/>
* OpenGraph e TwitterCard<br/>
* JPA/Hibernate<br/>
* MySQL<br/>
* Ajax<br/>
* Bootstrap<br/>
* XAMPP<br/>
